List out the different ways to select an item.

Answer: Different ways to select an item in CSS include element selectors, class selectors, ID selectors, attribute selectors, and descendant selectors.

What is the difference between an ID and a Class?

Answer: An ID is unique to one element while a class can be used on multiple elements.

What is the difference between a SPAN tag and a DIV tag?

Answer: A SPAN tag is an inline element for small pieces of content, while a DIV tag is a block element for larger sections.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

Answer: A DIV tag is a generic container without meaning, while HEADER, MAIN, and FOOTER are semantic tags that describe the structure of a webpage.

What are the different ways to select a color?

Answer: Colors in CSS can be selected using color names, HEX values, RGB/RGBA values, and HSL/HSLA values.
